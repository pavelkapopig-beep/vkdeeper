# VK Bot for Render

Простой чат‑бот для группы «ВКонтакте», развёрнутый на платформе Render.

## Функционал
- Отвечает на сообщение «привет» фразой «Здравствуйте! Чем могу помочь?».

## Настройка
1. Создайте группу «ВКонтакте» и получите `access_token` и `secret_key`.
2. Заполните переменные окружения на Render.
3. Настройте Callback API в настройках группы.

## Технологии
- Python 3.x
- Flask
- VK API
